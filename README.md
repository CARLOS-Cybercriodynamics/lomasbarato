# LoMásBarato.mx  
Comparador ligero de ofertas reales de supermercados en México.

## ¿Qué es esto?
Sitio web que muestra **ofertas verificadas semanalmente** de cadenas como Walmart, Soriana, Chedraui y Bodega Aurrerá.

## Actualización semanal
Los precios se actualizan cada lunes con un archivo `ofertas-semana.json`.

## Cómo contribuir
1. Actualiza el JSON.
2. Haz commit y push.
3. El sitio se actualiza automáticamente.

## Legal
Todos los precios provienen de información pública sin reproducción de material protegido o logos.
